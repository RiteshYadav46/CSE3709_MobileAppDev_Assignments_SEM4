## Project Overview
This repository contains the solution for the Even Semester 2026 Graded Assignment.

### Question 1: Currency Converter
- Converts between INR, USD, JPY, and EUR.
- Includes a Settings page for Light/Dark theme switching using SharedPreferences.

### Question 2: Media Player
- Plays local audio files from the device.
- Streams video from a URL.
- Features: Play, Pause, Stop, and Restart.

### Question 3: Sensor Monitor
- Real-time display of Accelerometer, Light, and Proximity sensor data.

### Question 4: Photo Gallery & Camera
- Custom camera integration to save photos to specific folders.
- Grid view gallery to browse images.
- Detailed view with image metadata (path, size, date) and delete functionality.

## How to Run
1. Clone this repository.
2. Open each folder as a separate project in Android Studio.
3. Build and run on an Emulator or Physical Device.
